# Add Brilliant Signature 3 font here

Place your **Brilliant Signature 3** font file in this folder and name it:

- `BrilliantSignature3.woff2` (preferred), or  
- `BrilliantSignature3.woff` or `BrilliantSignature3.ttf`

The main heading (Bibliotheca) will use this font. If the file is missing, the heading will use the default display font.
